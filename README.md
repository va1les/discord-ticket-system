# install
### 📨 **File** `config.json`
```js
{
    "token": "your Bot Token Here",
    "mongo_uri": "your Mongo Uri Here",
    "prefix": "!"
}
```

### 📥 Install all modules — `npm install`

### ✅ Launch the Bot — `node .`
